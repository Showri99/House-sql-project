# House Data Analysis using SQL

## Project Overview
Analyze a housing dataset using SQL. Demonstrates basic SQL queries including SELECT, WHERE, GROUP BY, ORDER BY, and aggregate functions.

## Dataset
- 50-row housing dataset with location, bedrooms, bathrooms, area, and price.

## Skills Demonstrated
- Creating database and tables
- Importing CSV data
- Aggregate functions (AVG, MIN, MAX)
- Filtering with WHERE
- Grouping with GROUP BY
- Sorting with ORDER BY

## How to Run
1. Open `house_data.db` in DB Browser for SQLite
2. Run queries from `house_analysis.sql`
3. Explore results
